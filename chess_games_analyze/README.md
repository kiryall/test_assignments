# Chess games analyze

**Модель машинного обучения для предсказания шахматных ходов на основе позиции на доске**

---

## О проекте

Разработана нейросетевая модель, способная предсказывать следующий ход в шахматной партии для любой возможной позиции. Модель использует современные подходы глубокого обучения с учетом специфики шахматных правил.

### Ключевые особенности
- **Многоголовая архитектура** для раздельной классификации исходной и целевой клеток
- **Система маскирования** легальных ходов для ускорения обучения
- **Поддержка всех шахматных правил** включая рокировку, превращения и взятие на проходе

---

## Архитектура решения

### Представление данных
- **Вход**: FEN-позиция → тензор 8×8×18
  - 12 плоскостей для типов фигур (6 белых + 6 черных)
  - 6 плоскостей для игрового состояния (очередь хода, рокировки, взятие на проходе)
- **Выход**: три целевых вектора
  - `from_target` (64) - исходная клетка
  - `to_target` (64) - целевая клетка
  - `promotion_target` (5) - тип превращения пешки

### Нейросетевая архитектура
```python
CNN Backbone → Multi-Head Output
    ↓
[from_head(64), to_head(64), promotion_head(5)]
```
