# Object Detection with YOLO

Этот проект предоставляет простой интерфейс для детектирования объектов на изображении с помощью моделей **Ultralytics YOLO**.

## Возможности

* Загрузка предобученной модели YOLO
* Детекция объектов на входном изображении
* Автоматическое сохранение изображения с визуализацией
* Подсчёт объектов выбранных классов

## Требования

Убедитесь, что установлены необходимые зависимости:

```bash
pip install -r requirements.txt
```

## Структура

* `MODEL_NAME` — название используемой модели (по умолчанию `yolo11n-seg.pt`)
* Основной скрипт выполняет:

  * парсинг аргументов
  * загрузку модели
  * выполнение детекции
  * сохранение результата
  * подсчёт объектов по указанным классам

## Запуск

Пример использования:

```bash
python detect_objects.py \
    --image path/to/image.jpg \
    --class person,car \
    --out_name out.jpg
```

## Вывод

Скрипт напечатает количество найденных объектов по каждому указанному классу, например:

```
Найдено 3 объектов класса 'person'
Найдено 1 объектов класса 'car'
```

## Аргументы

| Аргумент     | Описание                                  |
| ------------ | ----------------------------------------- |
| `--image`    | Путь к входному изображению               |
| `--class`    | Список классов для поиска (через запятую) |
| `--out_name` | Имя выходного файла с результатом         |

## Лицензия
